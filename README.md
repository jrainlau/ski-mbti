## 🎿 这是啥？

咱就是说，滑雪圈也该有自己的人格测试了吧？！

**滑雪佬 MBTI** 是一款专为滑雪爱好者打造的趣味人格测试 🧪

30 道精心设计的题目，覆盖你的技术偏好、装备态度、社交习惯、冒险精神和氛围感受——最终从 **16 种滑雪人格** 中，找到最像你的那一个 ✨

> 💡 评分体系参考了荣格（Carl Jung）分析心理学的类型理论，由心理学专家基于"对立面张力"和"功能分化"原则设计，10 个维度、双极量表评估，不是随便拍脑袋出来的哦～
